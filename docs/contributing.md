# Contributing to Tendril

For the full contribution guide, see [CONTRIBUTING.md](https://github.com/Trec-TorConsulting/Tendril/blob/main/CONTRIBUTING.md) in the repository root.

## Quick Start

1. **Fork** the repository
2. **Clone** your fork locally
3. **Create a branch** from `main` for your feature or fix
4. **Make changes** following the project conventions
5. **Submit a pull request** — all PRs require passing CI and one approval
