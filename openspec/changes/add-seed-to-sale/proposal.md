# Seed to Sale

## Summary
Full seed-to-sale tracking system — from genetics selection through harvest, curing, testing, and final sale/distribution. Compliance-ready for regulated markets.

## Status
**Planned** — future feature, not yet designed.

## Scope (Initial Ideas)
- Seed/clone intake tracking with genetics lineage
- Plant tagging (individual or batch) with unique identifiers
- Growth phase tracking with photo documentation
- Harvest logging (wet weight, dry weight, trim weight)
- Drying & curing tracking (humidity, temperature, duration)
- Lab testing integration (THC/CBD %, terpene profiles, contaminants)
- Package/lot creation with weight and labeling
- Transfer/sale logging with recipient tracking
- Waste/destruction logging
- Full audit trail for compliance reporting
- Barcode/QR generation for plant and package tags
- METRC / BioTrack / Leaf Data integration (state compliance APIs)

## Notes
- This is a large feature that touches nearly every part of the system
- Should be designed as an optional module (not all users are in regulated markets)
- Consider a phased rollout: basic tracking → compliance reporting → state API integration
